# FullCycle - Docker - Desafio 2

Para rodar a imagem localmente, siga as etapas a seguir, dentro da pasta em que estiver salvo o projeto em questão:

`docker-compose up -d`

Para acessar a aplicação:

`http://localhost:8080`
